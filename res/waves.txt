# 68000 Starfighters -- wave formation definitions.
#
# Human-editable source of truth for enemy waves. Parsed by
# generate_waves.py into src/waves_generated.h (compiled directly into the
# ROM) -- the Genesis has no runtime filesystem, so this file can't be read
# by the game itself; re-run the generator after editing it:
#   python generate_waves.py
#
# Fields per "wave N" block:
#   big <count>          number of BIG enemies in the back row (0-4)
#   grid <rows> <cols>   small-enemy grid dimensions
#   special <row> <col>  marks one grid cell (0-indexed) as a
#                        powerup-dropping SPECIAL enemy instead of a plain
#                        BEE; repeatable, any number per wave
#
# (rows*cols + big) must not exceed MAX_ENEMIES (30, see game.h). Waves are
# played in order and then loop back to wave 1, increasing the "waves
# cleared" counter forever.
